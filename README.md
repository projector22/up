# up

Perform an update on various linuxy systems and tools

## Update Types Available

### Application Managers

- Apt
- Nala (Apt frontend)
- Snap
- Flatpak
- Yum
- Pip3 - Only lists updates.

### Specific Applications

- PiHole
- rclone
